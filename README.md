# Features
- Events list with infinite loading
- Performers list
- View event info and book a ticket on seatgeek
- Search Performer or Event

# Technologies Used
- Expo (React Native)
- Redux, Redux Thunk, Redux Persist, React Native Dev Tools
- React Navigation
